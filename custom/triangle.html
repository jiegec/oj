1 题目回顾

输入一个 N ，输出前 N 层的杨辉三角。

1.1 数学基础

每一行除了第一项是1以外，每一项都是上一行的同一列和前一列上数字之和。

那我们就用数组实现这个简单问题吧：

首先是输入，然后初始化一个数组： 

#include <stdio.h>

int main() {
  int N;
  int row[20][20] = {0};
  scanf("%d",&N);
  return 0;
}

然后开始计算杨辉三角： 

row[0][0] = 1;
for (int i = 1;i <= N;i++) {
  row[i][0] = 1;
  printf("%d",row[i][0]);
  for (int j = 1;j < i;j++) {
    row[i][j] = row[i-1][j-1] + row[i-1][j];
    printf(" %d",row[i][j]);
  }
  putchar('\n');
 }
